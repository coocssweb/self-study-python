# Tech Stack

- Language: Python 3.8+ (uses features like positional-only parameters `/`)
- No external dependencies or third-party libraries
- Standard library only: `time`, `functools`
- No build system, package manager, or test framework

## Running Scripts

Each `.py` file is a standalone script. Run directly with:

```bash
python <folder>/<filename>.py
```

Example:
```bash
python 01/01_hello.py
python 02/01_def.py
```

There are no build, lint, or test commands configured for this project.
